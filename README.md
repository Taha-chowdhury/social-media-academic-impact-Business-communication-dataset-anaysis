# Social Media Academic Impact Analysis

## 📊 Project Overview
This project analyzes the relationship between social media usage and academic performance/concentration using statistical methods and data visualization techniques. It explores how digital behavior influences student productivity and cognitive focus.

The study is based on a structured dataset related to social media usage patterns, academic impact, sleep habits, and mental health indicators.

---

## 🎯 Objectives
- Analyze the correlation between social media usage and academic concentration
- Perform statistical hypothesis testing to validate relationships
- Measure effect size and significance of behavioral differences
- Visualize patterns using advanced data visualization techniques
- Extract meaningful insights for academic and behavioral understanding

---

## 🧰 Technologies Used
- Python 🐍
- Pandas, NumPy
- Matplotlib, Seaborn
- Plotly (interactive visualization)
- SciPy (statistical testing)
- Google Colab environment

---

## 📊 Key Methods
- Data Cleaning and Preprocessing
- Group-wise Statistical Comparison
- Independent T-test (significance testing)
- Effect size analysis (Cohen’s d)
- Correlation analysis
- Data visualization (boxplots, violin plots, heatmaps, pie charts, bar charts)

---

## 📈 Visualizations Included
- Box Plot comparison of usage groups
- Bar charts showing usage differences
- Pie charts for distribution analysis
- Violin plots for distribution shape comparison
- Stacked bar charts for categorical breakdown
- Correlation heatmap
- Interactive Plotly sunburst chart
- Summary statistical tables

---

## 🔍 Key Findings
- Students reporting concentration issues tend to have higher daily social media usage
- The difference in usage between groups is statistically significant (p < 0.05)
- Effect size indicates a meaningful behavioral impact
- Strong positive correlation observed between usage and academic impact
